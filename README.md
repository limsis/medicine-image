# 医学图像技术调研 - 基于SAM的眼底视网膜病变分割

## 项目简介

本项目针对医学图像分割领域，特别是眼底视网膜病变分割任务进行了深入调研。我们分析了Segment Anything Model (SAM)在医学图像分割中的应用，并针对两个核心问题提出了创新解决方案：

1. **问题一**：将通用医学图像分割模型迁移到特定的眼底视网膜分割任务
2. **问题二**：低质量眼底图像的分割性能下降问题

## 方法概述

### 1. RetinaSAM - 眼底视网膜专用SAM模型

我们提出了RetinaSAM，针对眼底图像特点进行了以下创新：

- **多尺度空洞空间金字塔池化 (ASPP)**：捕获不同大小的病灶
- **细长结构感知注意力**：借鉴裂缝检测技术，增强血管和微动脉瘤的特征
- **病灶上下文融合**：融合多尺度特征
- **改进的损失函数**：结合Dice损失、边界感知损失和骨架加权损失

### 2. EnhanceSegNet - 低质量图像增强与分割联合网络

为解决低质量图像分割问题，我们提出了EnhanceSegNet：

- **质量评估网络**：预测图像质量分数
- **自适应增强模块**：根据质量分数动态调整增强强度
- **联合训练策略**：同时优化图像增强和分割任务

## 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 训练RetinaSAM

```bash
cd src
python train.py --model retinasam --batch_size 8 --epochs 100 --lr 1e-4
```

### 训练EnhanceSegNet

```bash
cd src
python train.py --model enhancesegnet --batch_size 8 --epochs 100 --lr 1e-4
```

## 数据集

推荐使用以下公开的眼底图像数据集：

- **IDRiD**：糖尿病视网膜病变分割和分级挑战数据集
- **DDR**：糖尿病视网膜病变检测数据集
- **FGADR**：细粒度糖尿病视网膜病变数据集
- **Retina-lesion**：视网膜多病灶分割数据集

## 评估指标

- **Dice相似系数**：衡量预测与真实掩膜的重叠度
- **IoU (交并比)**：另一个常用的分割指标
- **灵敏度/召回率**：衡量病灶检测的完整性
- **精确率**：衡量预测的准确性
- **Hausdorff距离**：衡量边界的精确程度

## 创新点总结

1. **借鉴跨领域技术**：将裂缝检测和骨架提取的思想引入眼底图像分割
2. **专用特征适配器**：为眼底图像设计专用的特征提取模块
3. **质量感知增强**：根据图像质量自适应调整增强策略
4. **联合优化框架**：端到端训练图像增强和分割网络
