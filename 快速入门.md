## 💡 使用示例

### 示例1: 使用RetinaSAM
```python
import sys
sys.path.append('src')
from models.retinasam import RetinaSAM

# 创建模型
model = RetinaSAM(num_classes=5)

# 前向传播
import torch
x = torch.randn(1, 3, 512, 512)
output = model(x)  # 输出分割掩膜
```

### 示例2: 使用EnhanceSegNet
```python
from models.enhancesegnet import EnhanceSegNet

model = EnhanceSegNet(num_classes=5)

# 输入低质量图像
x = torch.randn(1, 3, 512, 512)
enhanced_img, mask, quality = model(x)
```

## 🎨 创新亮点

1. **RetinaSAM**: 针对眼底图像优化的SAM变体
   - 多尺度空洞卷积
   - 细长结构感知注意力
   - 改进的损失函数

2. **EnhanceSegNet**: 低质量图像增强+分割联合优化
   - 质量感知自适应增强
   - 端到端联合训练
